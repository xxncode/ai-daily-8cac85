人形机器人完成全球首例活体手术，用的是中国宇树 G1 平台

GPT-5.6 Agent 误删用户整台 Mac 文件，敲响 Agent 安全警钟

一位投资人反馈，其使用 GPT-5.6-Sol 的 AI Agent 在执行删除命令（rm）时，因错误展开了系统变量（HOME 变量指向根目录），把整台 Mac 的文件删了。

洞察　这是 Agent 从"聊天"走向"实际操作电脑"后必然暴露的风险——模型越自主、被授予的权限越大，一次判断失误的破坏力就越大。当前 Agent 落地真正的瓶颈不是"能不能干活"，而是"敢不敢把系统权限交给它"。可靠性、权限沙箱、可回滚，会成为下一阶段 Agent 产品的核心竞争力。

对你　给你个实用判断：谁能解决"让人放心把权限交给 Agent"，谁就握住了 Agent 大规模落地的钥匙。自己实测 Agent 时也务必在隔离环境里跑、别给生产数据的删改权限——既是安全，也是在体会这行最真实的痛点在哪。
